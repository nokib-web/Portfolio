## Thinking in React

React isn't just a library; it's a way of thinking about user interfaces. By breaking UIs into reusable components, we can build scalable and maintainable applications.

### 1. JSX: JavaScript XML
JSX allows us to write HTML-like code directly inside our JavaScript. It's syntactic sugar that ultimately compiles to `React.createElement`.

### 2. Components and Props
Components are the building blocks. Props are how we pass data into them.

```jsx
const Welcome = ({ name }) => {
  return <h1>Hello, {name}</h1>;
};
```

### 3. State Management with Hooks
The `useState` hook allows functional components to hold their own data.

```jsx
const [count, setCount] = useState(0);
```
