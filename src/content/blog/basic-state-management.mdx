## Managing Global State

As apps grow, passing props down 10 levels deep (Prop Drilling) becomes a nightmare. Use these tools to manage state centrally.

### 1. Context API
React's built-in solution. Best for low-frequency updates like Themes or User Authentication.

### 2. Redux Toolkit
The industry standard for large, complex applications. It provides a structured flow (Actions → Reducers → Store).

### 3. Zustand
The rising star. Minimalistic, fast, and does not require complex boilerplate like Redux.

```javascript
const useStore = create((set) => ({
  count: 0,
  increase: () => set((state) => ({ count: state.count + 1 })),
}))
```

---

## Which one to choose?
- Small app? **Context**
- Medium app? **Zustand**
- Enterprise? **Redux**
