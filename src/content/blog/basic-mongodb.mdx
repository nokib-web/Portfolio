## The Modern Database for Modern Apps

MongoDB is a document-oriented database. Instead of rows and columns, it stores data in JSON-like BSON documents.

### 1. Flexible Schema
Unlike SQL, you don't need to define a strict schema upfront. This makes MongoDB perfect for rapidly evolving projects.

### 2. CRUD Operations
- **Create:** `db.collection.insertOne()`
- **Read:** `db.collection.find()`
- **Update:** `db.collection.updateOne()`
- **Delete:** `db.collection.deleteOne()`

### 3. Mongoose ODM
Working with MongoDB in Node.js is usually done via Mongoose, providing schema validation and easy modelling.
