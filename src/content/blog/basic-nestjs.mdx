## Scaling Node.js to the Moon

NestJS is a framework for building efficient, reliable and scalable server-side applications. It's heavily inspired by Angular's architecture.

### 1. Modules, Controllers, Services
Nest enforces a modular structure. 
- **Controllers** handle incoming requests.
- **Services** contain the business logic.
- **Modules** organize the code.

### 2. Decorators
Nest makes heavy use of TypeScript decorators to define metadata.
```typescript
@Controller('users')
export class UsersController {
  @Get()
  findAll(): string {
    return 'Returns all users';
  }
}
```
